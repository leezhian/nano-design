# NavBar 导航栏

导航栏，主要适用于移动端。

## Props

|  属性   | 说明  | 类型  | 默认值  |
|  ----  | ----  |  ----  | ----  |
| title  | 将按钮宽度调整为其父宽度的选项 | boolean  | - |
| fixed  | 是否固定在顶部 | boolean  | false |
| ghost  | 是否是幽灵导航栏 | boolean  | false |
| extra  | 额外内容（显示在右侧） | ReactNode  | false |
| zIndex  | 层级 | number  | 1 |
| onLeftClick  | 点击左侧按钮时的回调 | (event) => any  | - |
